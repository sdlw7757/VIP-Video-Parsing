# VIP 视频解析播放器
一个基于 Python+Tkinter 开发的本地 VIP 视频解析播放器，支持主流视频网站链接解析，内置官网快捷跳转，操作简单且界面友好。

## 🚀 快速开始
### 环境依赖
项目已内置独立 Python 环境（Python文件夹），无需额外安装 Python，仅需确保：
- 项目根目录下的Python文件夹完整（包含python.exe和依赖库）；
- 依赖库requests已预装（项目内 Python 环境默认已包含）。

### 使用步骤
1. **克隆 / 下载项目**：将项目完整下载到本地任意磁盘（如D:\VIPVideoPlayer）；
2. **启动程序**：双击项目根目录下的Start_Silent.vbs，即可静默启动播放器（无黑窗口）；
3. **解析视频**：在 “请输入视频链接” 输入框中，右键粘贴视频网页链接；点击 “播放视频”，自动跳转到解析后的播放页面；
4. **官网跳转**：点击界面下方的 “腾讯视频”“爱奇艺” 等按钮，直接打开对应平台官网。

## 📋 功能特点
- 视频解析：输入主流视频网站链接，自动调用解析接口播放；
- 官网快捷跳转：一键打开腾讯视频、爱奇艺、优酷等平台官网；
- 便捷操作：输入框支持右键粘贴 / 复制 / 清空，操作更高效；
- 静默启动：通过 VBS 脚本一键启动，无黑窗口干扰；
- 跨路径运行：项目文件夹可放在任意磁盘，双击即可启动。
